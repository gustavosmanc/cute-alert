## Usage

```ts
import { cuteToast } from 'cute-alert'

cuteToast({
  type: 'success',
  title: 'Title',
  description: 'Description',
  timer: 5000
})
```

[Storybook](https://gustavosmanc.github.io/cute-alert/storybook/?path=/story/cutealert-toast--playground){: .btn }
